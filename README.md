
# Mohak – The Vibe | Firebase Studio

See Live: https://mohak-the-vibe.web.app/

This is the official repository for **Mohak – The Vibe**, a premium artificial jewelry boutique.

## Getting Started

1.  **Development**: All edits are made within the Firebase Studio IDE.
2.  **Saving Changes**: Changes are auto-saved to your development workspace.
3.  **Syncing to GitHub**: To push your changes to your connected repository, use the **"Sync with GitHub"** button in the Firebase Studio sidebar or header.
4.  **Deployment**:
    *   Click the **"Deploy"** button in the Firebase Studio UI to push your site live to Firebase Hosting.
    *   Your `firebase.json` is configured for Static Site Generation (SSG).

## Project Structure

*   `/src/app`: Next.js App Router pages and layouts.
*   `/src/components`: Reusable UI components (ShadCN).
*   `/src/lib`: Utility functions and image configurations.
*   `/public/images/earring`: All jewelry assets.
*   `src/lib/placeholder-images.json`: Manage all image paths here.

## Troubleshooting HMR
If you encounter Turbopack HMR errors, try re-saving the `src/lib/placeholder-images.json` file or clearing your browser cache.

For support, contact the Mohak dev team.
